Underground Cable Fault Detection System

Objective

Developed an efficient system for detecting and locating faults in
underground cables to minimize downtime and maintenance costs.

Approach

  • Utilized microcontrollers and sensors to measure the impedance and voltage variations along the cable.
  • Implemented a fault localization algorithm to pinpoint the exact location of the fault based on real-time data.
  • Designed a user-friendly interface to display the fault location and status information.

Technologies Used

  • Microcontrollers (Arduino/STM32)
  • Voltage and current sensors
  • Data processing and visualization software (MATLAB/LabVIEW)

Outcome

  • Successfully identified faults with high accuracy.
  • Reduced the time and effort required for fault detection and repair.
  • Enhanced reliability and efficiency of the underground cable network.
